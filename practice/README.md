# Practice Folder

This folder contains exercises and hands-on practice materials for prompt engineering.

## Structure

Exercises are organized by curriculum section:

```
practice/
├── 01-foundation/        # Core principles exercises
├── 02-architecture/      # Structural technique practice
├── 03-applied/           # Domain-specific tasks
├── 04-debugging/         # Optimization challenges
├── 05-systems/           # Workflow design exercises
├── 06-documentation/     # Library building practice
├── 07-cross-model/       # Model comparison exercises
└── 08-advanced/          # Complex technique practice
```

## How to Use

1. **Start with fundamentals** - Complete Section 1 exercises before moving forward
2. **Document your work** - Save your prompts and results in each exercise folder
3. **Iterate** - Try multiple versions of each prompt
4. **Reflect** - Note what worked and what didn't

## Exercise Format

Each exercise file follows this structure:

```markdown
# Exercise: [Name]

## Objective
[What skill this exercise develops]

## Task
[The specific challenge]

## Starter Prompt
[A basic prompt to improve]

## Your Work
[Space for your iterations]

## Reflection
[What you learned]
```

## Adding Your Own Exercises

Feel free to add custom exercises based on your real-world needs. Use the template above and place them in the appropriate section folder.
